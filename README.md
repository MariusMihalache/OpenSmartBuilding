## ✨ Funcționalități

* 🌡️ HVAC & AHU Monitoring
* ☀️ Solar Energy Dashboard
* ❄️ Chillers & Cooling Systems
* 🔥 Heating Plant Monitoring
* 💧 Pumps & Hydraulics
* 💡 Smart Lighting Control
* 🚨 Fire Detection Systems
* 🔓 Intrusion Detection
* 🪪 IP Access Control
* 🌤️ Weather Monitoring

---

## ⚡ Quick Start

```bash
git clone https://github.com/MariusMihalache/OpenSmartBuilding.git

cd OpenSmartBuilding
```

Deschide:

```bash
osb_enterprise_fixed.html
```

Nu necesită instalare, server sau dependențe.

* Modbus TCP
* Modbus RTU
* BACnet/IP
* MQTT
* REST API

*Versiunea actuală este un demo frontend și nu conține implementări backend sau conexiuni reale către echipamente BMS.*
